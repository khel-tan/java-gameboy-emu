# Gameboy Emulator
## Short description
An emulation of the original Gameboy console in Java.

## Features
- Emulation of the Sharp LR35902 that the original Gameboy uses
- Basic Memory Bus
- A test run of Bubble Sort to showcase that the CPU works

## Intended Features
- CPU interrupts
- Audio
- Gamepad
- Graphics
